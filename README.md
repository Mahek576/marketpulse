\# MarketPulse



MarketPulse is a production-grade market intelligence platform backend.



It is being built using FastAPI, PostgreSQL, Redis, background jobs, and AI-powered intelligence modules.



\## Current Version



v0.1.0



\## Current Features



\- FastAPI backend setup

\- Environment-based configuration using `.env`

\- Health check endpoint

\- Auto-generated API documentation using Swagger UI



\## Tech Stack



\- Python

\- FastAPI

\- Pydantic Settings

\- Uvicorn

\- PostgreSQL planned

\- Redis planned

\- Docker planned
